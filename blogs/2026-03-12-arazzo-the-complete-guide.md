Bump.sh published a comprehensive step-by-step guide to Arazzo, the OpenAPI Initiative standard for describing API workflows, with contributions from Phil Sturgeon. It explains how Arazzo chains multiple API calls together to accomplish business objectives, distinct from OpenAPI which documents individual API capabilities.
